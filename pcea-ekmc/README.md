# PCEA Elijah Kagiri Memorial Church - Spiritual Companion App

A modern, full-stack spiritual companion platform designed to enhance the faith journey of the PCEA Elijah Kagiri Memorial Church congregation through AI-powered insights, community engagement, and real-time worship.

## 🌟 Features

### 🎥 Live Worship Experience
- **Multi-Sanctuary Streaming**: Watch live services from both the Main Church and the Youth Sanctuary directly on the website.
- **Integrated Player**: No need to leave the site for YouTube or Facebook.
- **Real-time Status**: Live/Offline indicators for scheduled services.

### 📖 AI Sermon Insights
- **Automated Processing**: Analyze sermon transcripts directly in the browser.
- **Multilingual Summaries**: Get concise summaries in both English and Kiswahili.
- **3-Day Devotionals**: Personalized spiritual growth plans based on the latest sermon.
- **Discussion Questions**: 5 thought-provoking questions for small group study.

### 🤖 Bible AI Assistant
- **Biblical Guidance**: Ask theological questions or seek spiritual encouragement.
- **Context-Aware**: Grounded in scripture and the church's values.
- **No API Key Required**: Powered entirely by client-side browser AI, ensuring privacy and zero dependencies on external services.

### 🏫 Church Institutions
- **Ladies Hostels**: Safe and secure accommodation for young ladies and students, located within the Main Church Compound.
- **Memorial Academy**: Holistic education from **PP1 to Grade 9 (Junior Secondary)**, grounded in Christian values and located within the Main Church Compound.

### 🤝 Community & Growth
- **Group Registration**: Easy sign-up for church groups like Brigades, Women's Guild, and PCMF. Members provide their name, contact details, and **current estate** for better local coordination.
- **Prayer Wall**: Share prayer requests and join others in intercession.
- **Testimonies**: A dedicated space to share praise reports of God's goodness.
- **Service Schedule**: Clear visibility of all weekly services and timings.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS, Lucide React, Framer Motion.
- **Backend**: Node.js, Express.
- **AI**: Transformers.js (`@xenova/transformers`).
- **Storage**: Local JSON-based database for persistence.

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Set Up Environment**:
   No additional configuration is required to start the application.

3. **Run Development Server**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```
